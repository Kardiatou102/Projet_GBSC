# Analyse des disparités territoriales en France

## Problématique

Les départements français offrant les meilleures conditions de vie sont-ils aussi ceux aux revenus moyens les plus élevés, dotés de davantage d’infrastructures culturelles et de moins de logements sociaux ?

## Contexte

Ce projet vise à étudier les disparités territoriales en France à partir de facteurs socio-économiques, démographiques et culturels. L’objectif est d’identifier les principaux déterminants de la qualité de vie au niveau départemental.
Le projet a été réalisé par quatre personnes.

## Objectifs

- Analyser les relations entre revenus, chômage, pauvreté, établissements culturels et logements sociaux
- Étudier l’impact des infrastructures culturelles sur l’attractivité des territoires
- Identifier des corrélations significatives entre variables socio-économiques
- Fournir une interprétation des dynamiques territoriales

## Méthodologie

- Collecte et sélection de données issues de différentes sources
- Nettoyage et harmonisation des données
- Modélisation de la base de données
- Analyses statistiques (corrélations, ANOVA)
- Visualisation des résultats
- Interprétation socio-économique

## Compétences mobilisées

- Conception de base de données (MCD)

- Manipulation de données avec R

## Résultats clés

- Absence de corrélation directe entre le niveau de revenu et les taux de chômage ou de pauvreté
- Existence d’un lien significatif entre le revenu moyen et le nombre d’établissements culturels (test ANOVA)
- Corrélation positive entre le nombre d’infrastructures culturelles et le taux de logements sociaux
